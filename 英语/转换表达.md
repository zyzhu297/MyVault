# 转换表达

写作/口语中，将普通词汇替换为更精准、更高级的表达，提升语言质量。

## 动词转换

| 普通表达      | 高分表达                                           | 例句                                                             |
| --------- | ---------------------------------------------- | -------------------------------------------------------------- |
| show      | depict / portray / illustrate                  | The data **illustrates** a clear upward trend.                 |
| think     | believe / hold the view that / argue / contend | Many scholars **contend** that early intervention is key.      |
| solve     | tackle / address / resolve                     | We must **tackle** this issue at its root.                     |
| face      | embrace / confront / encounter                 | She decided to **embrace** the challenge head-on.              |
| improve   | enhance / refine / elevate / boost             | The new policy will **enhance** overall efficiency.            |
| help      | contribute to / facilitate / assist            | Exercise **contributes to** better mental health.              |
| start     | commence / embark on / initiate                | The team decided to **embark on** a new project.               |
| get       | obtain / acquire / gain / secure               | Students can **acquire** practical skills through internships. |
| give      | provide / offer / supply / deliver             | The program **provides** valuable hands-on experience.         |
| make      | create / produce / generate / establish        | We aim to **establish** a solid foundation for growth.         |
| use       | utilize / employ / leverage / harness          | We should **harness** renewable energy sources.                |
| change    | transform / modify / alter / shift             | Technology has **transformed** the way we communicate.         |
| need      | require / necessitate / call for               | This approach **requires** careful planning.                   |
| cause     | lead to / result in / trigger / bring about    | Stress can **trigger** a range of health problems.             |
| end       | conclude / terminate / cease                   | The study **concluded** that the drug was effective.           |
| find      | discover / identify / detect / uncover         | Researchers **identified** three key factors.                  |
| say       | state / assert / claim / remark                | The author **asserts** that education is a fundamental right.  |
| look at   | examine / inspect / analyze / investigate      | Let's **examine** the evidence more closely.                   |
| try       | attempt / endeavor / strive to                 | We must **strive to** achieve a better balance.                |
| believe   | be convinced that / hold the conviction that   | I am **convinced that** hard work pays off.                    |
| influence | exert an influence on / shape / mold           | Parents **shape** a child's early development.                 |
| develop   | cultivate / foster / nurture                   | Teachers should **cultivate** students' curiosity.             |

## 使用建议

1. **不要过度堆砌**：一篇好文章是自然流畅的，不必每个词都换成"高级"表达。关键是**精准**而非**华丽**。
2. **语境匹配**：`enormous` 适合形容数量，`vital` 适合形容重要性，选词要看搭配。
3. **正式度一致**：如果你用 `kids`，就别在同一段用 `offspring`；保持全文语域统一。
4. **动词是灵魂**：相比形容词和副词，精准的动词更能提升文章档次。
5. **多读多用**：最好的方法是在阅读中积累，在写作中刻意练习。
