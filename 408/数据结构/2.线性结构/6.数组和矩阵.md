# 数组的定义
由$n$个相同类型的数据元素组成的有限序列，每个元素称为**数组元素**，其在线性序列中的位置由*下标*标识，下标的取值范围称为**维界**
## 数组和线性表的关系
数组是线性表的推广，一维数组可视为一个线性表，二维数组可视为其元素为**定长一维数组**的线性表
数组一旦定义，其维数和维界即固定不变
# 数组的基本操作
除初始化和销毁外，数组仅支持存取元素和修改元素两种基本操作
# 数组的存储结构
逻辑上的数组**映射为内存中一段连续的存储空间**
设每个元素占 $L$ 个储存单元
- 一维数组 A$[1,2,\dots,n-1]$ ，其中 A$[i]$ 的地址为$$LOC(a_i) = LOC(a_0) + iL (0\le i < n)$$
- 多维数组在内存需按某种顺序展开为一维，设二维数组行下标与列下标范围分别为 $[0,h_1]$ 和 $[0,h_2]$ ，则元素 A$[i][j]$ 的地址为
	- 按行优先：先存储行号较小的元素；同一行内先存储列号较小的元素$$LOC(a_{i,j}) = LOC(a_{0,0}) + [i(h_2+1) + 1]L$$
	- 按列优先：先存储列号较小的元素；同一列内先存储行号较小的元素$$LOC(a_{i,j}) = LOC(a_{0,0}) + [j(h_1+1) + 1]L$$
# 特殊矩阵的压缩存储
- 压缩存储：为多个值相同的元素只分配一个存储空间，对零元素不分配存储空间
## 对称矩阵
将$n$阶对称矩阵$A$压缩存储于一维数组 B$[n(n + 1)/2]$中，通常仅存放下三角部分（含主对角线）
数组下标从 0 开始，元素 $A[i][j]$ 在一维数组 $B$ 中的下标为 $k$：
$$k = \begin{cases}
\frac{i(i-1)}{2} + j - 1, & i \ge j （下三角区和主对角元素）\\
\frac{j(j-1)}{2} + i - 1, & i < j （上三角区元素a_{i,j}=a_{j,i}）
\end{cases}$$
## 三角矩阵
以下都是行优先顺序存储于数组的情况
### 下三角矩阵
上三角区的所有元素均为**同一常量**，其存储思想和对称矩阵类似，但是需要额外存储该常量一次，将 $n$ 阶三角矩阵$A$压缩存储于一维数组 B$[n(n + 1)/2+1]$中
- 第1行：$1$ 个元素
- 第$i-1$行：$i-1$个元素
- 第$i$行：$j-1$个元素
数组下标从 0 开始，元素 $A[i][j]$ 在一维数组 $B$ 中的下标为 $k$：
$$\large k = \begin{cases}
\frac{i(i-1)}{2} + j - 1, & i \ge j \ \text{(下三角区和主对角元素)}\\
\frac{n(n+1)}{2}, & i < j \ \text{(常量存储在数组的最后一个位置)}
\end{cases}$$
### 上三角矩阵
- 第1行：$n$ 个元素
- 第$i-1$行：$j-i+2$个元素
- 第$i$行：$j-i$个元素
$$\large k = \begin{cases}
\frac{(2n-i+2)(i-1)}{2} + j - i, & i \le j\ \text{(上三角区和主对角元素)}\\
\frac{n(n+1)}{2}, & i > j\ \text{(常量存储在数组的最后一个位置)}
\end{cases}$$
## 三对角矩阵
若 $n$ 阶矩阵中的任意元素 $a_{i,j}$，当 $|i-j|>1$ 时均有 $a_{i,j}=0(1 \le i,j \le n)$，则称为三对角矩阵
将 $n$ 阶三对角矩阵 $A$ 压缩存储于一维数组 B$[3n-2]$ 中，仅存储三条对角线上的元素
数组下标从 0 开始，元素 $A[i][j]$ 在一维数组 $B$ 中的下标为 $k$：
$$k = 2i + j - 3, \quad |i - j| \le 1$$

已知 $k$ 时，可反推 $i, j$（$0 \le k < 3n-2$）：
$$i = \left\lfloor \frac{k+1}{3} \right\rfloor + 1, \quad j = k - 2i + 3$$
## 稀疏矩阵
若矩阵中的非零元素个数$t$相对于总元素个数 $s$ 来说非常少，即 $t \ll s$，则称该矩阵为*稀疏矩阵*
为避免空间浪费，稀疏矩阵通常仅存储非零元素，为此将每个非零元素及其对应的行列位置组合成一个**三元组**（$行标i，列标j，值a_{i,j}$）
三元组可以通过以下方式存储：
- 数组存储：按某种顺序存储在一维数组中，简单直接，但插入和删除操作效率低
- 十字链表存储：适用于频繁插入和删除的场景
> 注意：无论采用何种方式都要额外保存稀疏矩阵的**行数**、**列数**和**非零元素个数**
