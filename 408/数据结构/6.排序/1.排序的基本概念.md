# 排序的定义
将一组任意排列的**数据元素**，按照某个关键字的值递增（或递减）重新排列，使其成为有序序列的过程
- **关键字(Key)**：数据元素中用于排序的某个数据项的值
  - **主关键字**：关键字值唯一标识一个元素（如学号）
  - **次关键字**：关键字值可重复（如年龄、成绩）
- 排序结果：若按递增排列，则称**升序**（非递减）；按递减排列，则称**降序**（非递增）
> 数据结构中，通常默认讨论**升序排序**
> 排序只关注关键字，不同的排序算法面对有相同关键字的序列结果可能不同，取决于算法是否*稳定*
# 排序的稳定性
**定义**：若待排序序列中存在两个关键字相等的元素 $R_i$ 和 $R_j$（$i < j$），排序后 $R_i$ 仍然在 $R_j$ 之前（**即相对位置**），则称该排序算法是**稳定的**；否则称为**不稳定**的
> 证明不稳定只需要举反例
# 内部排序与外部排序
- 待排序记录全部存放在内存中进行排序称为**内部排序**
- 待排序记录数量太大，需要借助外存（磁盘）进行排序称为**外部排序**
# 排序算法的性能评价指标
## 时间复杂度
- **最好情况**、**最坏情况**、**平均情况**下的比较次数和移动次数
- 直接决定算法在大规模数据下的适用性
## 空间复杂度
- 衡量排序过程中需要的辅助存储空间
- **原地排序(In-place)**：空间复杂度为 $O(1)$ 的排序算法
> 稳定性于算法优劣**无关**
# 排序算法的分类（按实现策略）

| 类别       | 代表算法                  |
| -------- | --------------------- |
| **插入排序** | 直接插入排序、折半插入排序、希尔排序    |
| **交换排序** | 冒泡排序、快速排序             |
| **选择排序** | 简单选择排序、堆排序            |
| **归并排序** | 二路归并排序、多路归并排序         |
| **基数排序** | 最高位优先（MSD）、最低位优先（LSD） |

# 各排序算法对比一览

| 排序算法   |    平均时间复杂度     |   最坏时间复杂度    |   最好时间复杂度    |    空间复杂度    | 稳定性 |
| ------ | :------------: | :----------: | :----------: | :---------: | :-: |
| 直接插入排序 |    $O(n^2)$    |   $O(n^2)$   |    $O(n)$    |   $O(1)$    | ✅   |
| 折半插入排序 |    $O(n^2)$    |   $O(n^2)$   | $O(n\log n)$ |   $O(1)$    |  ✅  |
| 希尔排序   | $O(n^{1.3-2})$ |   $O(n^2)$   |    $O(n)$    |   $O(1)$    |  ❌  |
| 冒泡排序   |    $O(n^2)$    |   $O(n^2)$   |    $O(n)$    |   $O(1)$    |  ✅  |
| 快速排序   |  $O(n\log n)$  |   $O(n^2)$   | $O(n\log n)$ | $O(\log n)$ |  ❌  |
| 简单选择排序 |    $O(n^2)$    |   $O(n^2)$   |   $O(n^2)$   |   $O(1)$    |  ❌  |
| 堆排序    |  $O(n\log n)$  | $O(n\log n)$ | $O(n\log n)$ |   $O(1)$    |  ❌  |
| 归并排序   |  $O(n\log n)$  | $O(n\log n)$ | $O(n\log n)$ |   $O(n)$    |  ✅  |
| 基数排序   |  $O(d(n+r))$   | $O(d(n+r))$  | $O(d(n+r))$  |   $O(r)$    |  ✅  |

> 注：$n$ 为元素个数，$d$ 为关键字位数，$r$ 为基数（如十进制 $r=10$）。
# 相关术语

| 术语            | 说明                                      |
| ------------- | --------------------------------------- |
| **有序区 / 无序区** | 排序过程中序列分成两部分，前部已有序，后部待排序                |
| **比较排序**      | 基于关键字比较运算的排序，最坏情况下比较下界为 $O(n\log n)$    |
| **非比较排序**     | 基数排序、计数排序、桶排序，不依赖比较，可突破 $O(n\log n)$ 下界 |
| **一趟（pass）**  | 排序算法的一次基本处理过程                           |
| **原址排序**      | 空间复杂度 $O(1)$ 的排序算法                      |
