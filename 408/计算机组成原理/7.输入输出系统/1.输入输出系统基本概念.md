# 输入/输出系统
## 基本概念

**I/O 系统** 是计算机中负责主机与外部世界进行数据交换的子系统，由 **I/O 软件** 和 **I/O 硬件** 协同构成。

- **外部设备**：输入设备、输出设备、外存储器等物理实体。
- **接口**：主机与外设之间的桥梁，实现数据缓冲、信号转换、设备选择等功能。
- **输入设备**：将外部信息（文字、图像、声音等）转换为计算机可识别的二进制数据。
- **输出设备**：将计算机处理结果转换为人或其他设备可识别的形式。
- **外存设备**：辅助存储器，用于长期保存大量数据和程序。

**I/O 设备的编址方式**：

| 编址方式 | 特点 | 优点 | 缺点 |
|---------|------|------|------|
| **统一编址（存储器映射）** | I/O 端口与主存统一编址，用访存指令访问 I/O | 指令丰富，编程灵活 | 占用主存地址空间；译码电路复杂 |
| **独立编址（I/O 映射）** | I/O 端口有独立地址空间，用专用 I/O 指令 | 不占主存空间；译码简单 | I/O 指令功能有限 |

## I/O 软件

I/O 软件主要包括设备驱动程序、I/O 管理程序以及用户应用程序；采用 **层次结构** 设计，自顶向下分为四层：

| 层次 | 功能 |
|------|------|
| **用户层 I/O 软件** | 提供用户程序调用的 I/O 库函数（如 `printf`、`scanf`） |
| **设备独立性软件** | 统一管理所有设备，提供命名、保护、缓冲、分配等功能 |
| **设备驱动程序** | 直接控制设备控制器，将上层通用请求转换为设备特定操作 |
| **中断处理程序** | 响应设备中断，保存现场、处理中断、恢复现场并返回 |

> **设计原则**：底层对上层隐藏硬件细节，每一层只与相邻层交互。操作系统核心负责中间两层的实现。

## I/O 硬件

I/O 硬件包括外部设备、设备控制器、I/O 接口以及 I/O 总线；从 CPU 侧到设备侧依次包含：

| 组成部分 | 说明 |
|---------|------|
| **I/O 端口（寄存器）** | CPU 与外设通信的接口寄存器，分为 **数据寄存器**、**状态寄存器**、**控制寄存器** |
| **I/O 接口** | 位于总线和外设之间，完成数据缓冲、格式转换、设备选择、中断控制等功能（详见 [[2.输入输出接口]]） |
| **设备控制器** | 控制一个或多个同类设备，接收 CPU 命令并控制设备操作 |
| **通道** | 专用的 I/O 处理器，可独立执行通道程序，管理多个设备的 I/O 操作 |


# 外部设备

## 输入设备

| 设备 | 基本原理 |
|------|---------|
| **键盘** | 按键 → 键码扫描矩阵 → 编码器 → ASCII 码 → 主机 |
| **鼠标** | 光电/机械式传感器检测位移变化 → 坐标增量信号 → 主机 |

## 输出设备

### 显示器

**（1）分类**

| 类型 | 代表 |
|------|------|
| CRT 显示器 | 阴极射线管，逐渐淘汰 |
| LCD 显示器 | 液晶显示器，主流 |
| LED 显示器 | 发光二极管 |

**（2）主要性能指标**
- **屏幕尺寸**：以对角线长度表示，单位为英寸
- **分辨率**：屏幕像素点数（如 1920×1080）
- **刷新机制**：由于像素发光持续时间极短，必须周期性重绘整屏图像，以维持画面稳定
- **刷新频率**：每秒刷新屏幕次数，≥ 60Hz 人眼才不会感到闪烁
- **灰度级/色彩深度**：每个像素可显示的颜色种类
- **显示存储器（VRAM）容量**：
  - 容量 = 分辨率 × 颜色深度（位宽）
  - 例：1024×768 × 24bit = 1024×768×3B ≈ 2.25MB

### 打印机

| 类型 | 原理 | 特点 |
|------|------|------|
| **针式打印机** | 打印针撞击色带 | 可复写打印，速度慢，噪声大 |
| **喷墨打印机** | 喷嘴喷出墨滴 | 分辨率高，色彩好，速度一般 |
| **激光打印机** | 激光扫描 + 静电成像 | 速度快，质量高，主流办公打印机 |

## 外存储器

### 磁表面存储器（磁盘）

**（1）存储原理**

利用磁头在磁介质表面进行电磁转换来读写数据。

**（2）基本参数**

| 参数 | 含义 |
|------|------|
| **磁道** | 盘面上的同心圆环，数据存储的基本路径 |
| **柱面** | 所有盘面上相同编号的磁道构成的圆柱面 |
| **扇区** | 磁道上划分的弧形段，通常为 512B |
| **记录密度** | 道密度（径向）+ 位密度（切向） |
| **存储容量** | 面数 × 每面磁道数 × 每道扇区数 × 每扇区字节数 |
| **平均存取时间** | 寻道时间 + 旋转延迟 + 传输时间 |

> **平均存取时间** = 平均寻道时间（机械臂移动到目标磁道）+ 平均旋转延迟（等待扇区转到磁头下，一般为旋转半周的时间）+ 数据传输时间

**（3）磁盘地址**

磁盘地址由 **柱面号（磁道号）→ 磁头号（面号）→ 扇区号** 组成，三维寻址。

### 固态硬盘（SSD）

| 特性 | 说明 |
|------|------|
| 存储介质 | NAND Flash 闪存芯片 |
| 基本单元 | 页（Page，如 4KB）、块（Block，多个页组成） |
| **读/写单位** | 页（Page） |
| **擦除单位** | 块（Block） |
| 特点 | 无机械部件，读写速度快，功耗低；**写前需擦除**，有擦写寿命限制 |

> **SSD 与 HDD 关键区别**：
> - HDD 可原地写覆盖；SSD 必须先擦除再写入。
> - SSD 通过 **FTL（闪存转换层）** 实现地址映射和磨损均衡。

### 光盘存储器

| 类型 | 特点 |
|------|------|
| CD-ROM | 只读，容量约 700MB |
| DVD-ROM | 只读，单面单层 4.7GB |
| CD-RW / DVD-RW | 可擦写 |

利用激光照射介质表面，通过反射光强弱读取数据（坑-岸方式）。

## 重要计算汇总

| 计算项 | 公式 |
|--------|------|
| VRAM 容量 | 分辨率 × 颜色位数（bit） |
| 磁盘容量 | 面数 × 磁道数/面 × 扇区数/道 × 字节数/扇区 |
| 平均存取时间 | T = T_seek（寻道）+ T_rotation（旋转延迟）+ T_transfer（传输） |
| 数据传输率 | 磁道容量 × 转速（转/s）|
