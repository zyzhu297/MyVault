# 计算机硬件发展

## 四代计算机

| 代次  | 逻辑元件         | 主存储器      | 特点                              |
| --- | ------------ | --------- | ------------------------------- |
| 第一代 | 电子管          | 汞延迟线、磁鼓   | 体积大、功耗高、速度慢，使用机器语言              |
| 第二代 | 晶体管          | **磁芯**存储器 | 体积减小、功耗降低、速度提升，出现高级语言(如FORTRAN) |
| 第三代 | 中小规模集成电路     | 半导体存储器    | 体积更小、功耗更低、速度更快，出现操作系统           |
| 第四代 | 大规模/超大规模集成电路 | 半导体存储器    | 微处理器诞生，个人计算机普及                  |

> [!note]+ 关键标志
> - 第一代代表：**ENIAC**（世界上第一台通用电子计算机，1946年，冯·诺依曼存储程序思想）
> - 第四代标志：**微处理器**（1971年Intel 4004）

## 摩尔定律

> **摩尔定律**（Moore's Law）：集成电路上可容纳的晶体管数目，约每隔**18-24个月**便会增加一倍，性能也将提升一倍。

- 并非自然科学定律，而是对技术发展趋势的观察与预测

# 计算机软件发展

## 程序设计语言发展阶段

| 阶段   | 语言类型    | 特点                                |
| ---- | ------- | --------------------------------- |
| 机器语言 | 二进制代码   | 唯一可直接被计算机执行的语言，但可读性差、不易编程         |
| 汇编语言 | 符号化指令   | 用助记符代替二进制，需汇编程序翻译，与机器一一对应         |
| 高级语言 | 面向过程/对象 | 如FORTRAN、C、C++、Java，接近自然语言，需编译/解释 |

**翻译程序分类：**
- **汇编程序**（Assembler）：汇编语言 → 机器语言
- **解释程序**（Interpreter）：逐条翻译并执行（如Python解释器）
- **编译程序**（Compiler）：整体翻译成目标程序再执行（如C编译器）

## 操作系统发展阶段

1. **手工操作阶段**：人工插拔线路，资源利用率极低
2. **批处理系统**：单道批处理 → 多道批处理，提高吞吐量
3. **分时操作系统**：多用户交互式使用，代表：UNIX
4. **实时操作系统**：对外部事件及时响应，用于工业控制等
5. **现代操作系统**：Windows、Linux、macOS，支持多任务、网络、图形界面
