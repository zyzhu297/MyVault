# 函数的四种性态

函数的性态（也称**特性**）是研究函数基本行为的四个维度，贯穿整个高等数学。

---

## 1. 有界性 (Boundedness)

> 若存在正数 $M$，使得对于定义域内所有 $x$，都有 $|f(x)| \leq M$，则称 $f(x)$ 为**有界函数**。

| 类型 | 定义 |
|------|------|
| **有上界** | 存在 $K$，使得 $f(x) \leq K$ |
| **有下界** | 存在 $K$，使得 $f(x) \geq K$ |
| **有界** | 既有上界又有下界，即 $\|f(x)\| \leq M$ |

**常见例子：**
- 有界函数：$\sin x$、$\cos x$、$\arctan x$
- 无界函数：$x^2$（有下界无上界）、$\tan x$、$e^x$（有下界无上界）

> [!important] 极限与有界
> 收敛数列必有界；但有界数列不一定收敛（如 $(-1)^n$）。

---

## 2. 单调性 (Monotonicity)

> 对于定义域内任意 $x_1 < x_2$，比较 $f(x_1)$ 与 $f(x_2)$ 的大小关系。

| 类型 | 条件 |
|------|------|
| **单调递增** | $x_1 < x_2 \Rightarrow f(x_1) \leq f(x_2)$ |
| **严格单调递增** | $x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$ |
| **单调递减** | $x_1 < x_2 \Rightarrow f(x_1) \geq f(x_2)$ |
| **严格单调递减** | $x_1 < x_2 \Rightarrow f(x_1) > f(x_2)$ |

**判别方法：**
- **导数法**：$f'(x) \geq 0 \Rightarrow$ 单调递增；$f'(x) \leq 0 \Rightarrow$ 单调递减
- **定义法**：构造 $f(x_2) - f(x_1)$ 并判断符号

> [!tip] 单调性的关键应用
> - 单调函数必有反函数（在严格单调区间上）
> - 单调有界定理：单调有界数列必收敛
> - 证明不等式：利用单调性确定函数值的范围

---

## 3. 奇偶性 (Odd-Even Property / Symmetry)

> 比较 $f(-x)$ 与 $f(x)$（以及 $-f(x)$）的关系。

| 类型 | 条件 | 图像特征 |
|------|------|----------|
| **偶函数** | $f(-x) = f(x)$ | 关于 $y$ 轴对称 |
| **奇函数** | $f(-x) = -f(x)$ | 关于原点对称 |

**常见例子：**

| 偶函数 | 奇函数 |
|--------|--------|
| $x^2$、$x^{2n}$ | $x$、$x^{2n-1}$ |
| $\cos x$ | $\sin x$、$\tan x$ |
| $\|x\|$ | $\arcsin x$、$\arctan x$ |
| $e^x + e^{-x}$ | $e^x - e^{-x}$ |

**运算性质：**

| 运算 | 结果 |
|------|------|
| 奇 ± 奇 | 奇 |
| 偶 ± 偶 | 偶 |
| 奇 × 奇 | 偶 |
| 偶 × 偶 | 偶 |
| 奇 × 偶 | 奇 |
| 奇 / 奇 | 偶 |
| 偶 / 奇 | 奇 |

> [!tip] 重要结论
> - 任何定义在对称区间上的函数 $f(x)$ 可唯一分解为奇函数与偶函数之和：
>   $$f(x) = \underbrace{\frac{f(x) - f(-x)}{2}}_{\text{奇函数}} + \underbrace{\frac{f(x) + f(-x)}{2}}_{\text{偶函数}}$$
> - 若 $f(x)$ 为奇函数且在 $x=0$ 处有定义，则 $f(0) = 0$
> - 奇函数在对称区间上的定积分为零：$\int_{-a}^{a} f(x)\,dx = 0$

---

## 4. 周期性 (Periodicity)

> 若存在非零常数 $T$，使得对于定义域内所有 $x$，有 $f(x+T) = f(x)$，则称 $f(x)$ 为**周期函数**，$T$ 为一个周期。

**最小正周期**：所有正周期中最小的那个（如果存在）。

**常见周期函数：**

| 函数                              | 周期                         |
| ------------------------------- | -------------------------- |
| $\sin x$、$\cos x$               | $2\pi$                     |
| $\tan x$、$\cot x$               | $\pi$                      |
| $\sin \omega x$、$\cos \omega x$ | $T = \dfrac{2\pi}{\omega}$ |
| $\tan \omega x$、$\cot \omega x$ | $T = \dfrac{\pi}{\omega}$  |

> [!warning] 注意
> - 并非所有周期函数都有最小正周期。例如常函数 $f(x) = C$，任意非零实数都是其周期，但不存在最小的。
> - 两个周期函数的和不一定是周期函数（除非它们的周期可公度）。
