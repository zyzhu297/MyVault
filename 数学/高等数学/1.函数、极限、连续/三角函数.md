
# 正割函数

$$y = \sec x = \frac{1}{\cos x}$$

## 基本性质

- **定义域**：$x \neq \frac{\pi}{2} + k\pi$
- **值域**：$(-\infty, -1] \cup [1, +\infty)$
- **奇偶性**：**偶函数**
- **周期性**：$T = 2\pi$
- **关键关系**：$\sec^2 x = 1 + \tan^2 x$

---

# 余割函数

$$y = \csc x = \frac{1}{\sin x}$$

## 基本性质

- **定义域**：$x \neq k\pi$
- **值域**：$(-\infty, -1] \cup [1, +\infty)$
- **奇偶性**：**奇函数**
- **周期性**：$T = 2\pi$
- **关键关系**：$\csc^2 x = 1 + \cot^2 x$

---

# 反三角函数

## 反正弦

$$y = \arcsin x$$

- **定义域**：$[-1, 1]$
- **值域**：$[-\frac{\pi}{2}, \frac{\pi}{2}]$
- **奇函数**：$\arcsin(-x) = -\arcsin x$
- **单调递增**

## 反余弦

$$y = \arccos x$$

- **定义域**：$[-1, 1]$
- **值域**：$[0, \pi]$
- **单调递减**
- 恒等式：$\arcsin x + \arccos x = \frac{\pi}{2}$

## 反正切

$$y = \arctan x$$

- **定义域**：$(-\infty, +\infty)$
- **值域**：$(-\frac{\pi}{2}, \frac{\pi}{2})$
- **奇函数**：$\arctan(-x) = -\arctan x$
- **单调递增**，有水平渐近线 $y = \pm\frac{\pi}{2}$

## 反余切

$$y = \operatorname{arccot} x$$

- **定义域**：$(-\infty, +\infty)$
- **值域**：$(0, \pi)$
- **单调递减**
- 恒等式：$\arctan x + \operatorname{arccot} x = \frac{\pi}{2}$

---

# 三角函数与反三角函数的复合

> 核心思路：画直角三角形，利用勾股定理。

## 一、直接互逆（定义域内）

$$\sin(\arcsin x) = x \quad (|x| \leq 1)$$
$$\cos(\arccos x) = x \quad (|x| \leq 1)$$
$$\tan(\arctan x) = x \quad (x \in \mathbb{R})$$
$$\cot(\operatorname{arccot} x) = x \quad (x \in \mathbb{R})$$

以及反过来：

$$\arcsin(\sin x) = x \quad \left(x \in \left[-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right]\right)$$
$$\arccos(\cos x) = x \quad \bigl(x \in [0, \pi]\bigr)$$
$$\arctan(\tan x) = x \quad \left(x \in \left(-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right)\right)$$

> **注意**：反函数在外层时，$x$ 的取值受值域限制，超出范围需要先用诱导公式调整。

---

## 二、不同名复合（画直角三角形）

- $\sin(\arcsin x) = x$，定义域 $|x| \leq 1$
- $\cos(\arccos x) = x$，定义域 $|x| \leq 1$
- $\tan(\arctan x) = x$，定义域 $x \in \mathbb{R}$
- $\sin(\arccos x) = \sqrt{1-x^2}$，定义域 $|x| \leq 1$
- $\cos(\arcsin x) = \sqrt{1-x^2}$，定义域 $|x| \leq 1$
- $\tan(\arcsin x) = \dfrac{x}{\sqrt{1-x^2}}$，定义域 $|x| < 1$
- $\tan(\arccos x) = \dfrac{\sqrt{1-x^2}}{x}$，定义域 $0 < |x| \leq 1$
- $\sin(\arctan x) = \dfrac{x}{\sqrt{1+x^2}}$，定义域 $x \in \mathbb{R}$
- $\cos(\arctan x) = \dfrac{1}{\sqrt{1+x^2}}$，定义域 $x \in \mathbb{R}$
- $\sin(\operatorname{arccot} x) = \dfrac{1}{\sqrt{1+x^2}}$，定义域 $x \in \mathbb{R}$
- $\cos(\operatorname{arccot} x) = \dfrac{x}{\sqrt{1+x^2}}$，定义域 $x \in \mathbb{R}$

# 常见三角变换公式

## 一、同角三角函数基本关系

### 平方关系

$$\sin^2 \alpha + \cos^2 \alpha = 1$$
$$1 + \tan^2 \alpha = \sec^2 \alpha$$
$$1 + \cot^2 \alpha = \csc^2 \alpha$$

### 倒数关系

$$\tan \alpha \cdot \cot \alpha = 1$$
$$\sin \alpha \cdot \csc \alpha = 1$$
$$\cos \alpha \cdot \sec \alpha = 1$$

### 商数关系

$$\tan \alpha = \frac{\sin \alpha}{\cos \alpha}$$
$$\cot \alpha = \frac{\cos \alpha}{\sin \alpha}$$

---

## 二、诱导公式

> **口诀**：奇变偶不变，符号看象限。

### $-\alpha$（负角公式）

$$\sin(-\alpha) = -\sin\alpha,\quad \cos(-\alpha) = \cos\alpha$$
$$\tan(-\alpha) = -\tan\alpha,\quad \cot(-\alpha) = -\cot\alpha$$

### $\frac{\pi}{2} \pm \alpha$

$$\sin\left(\frac{\pi}{2} - \alpha\right) = \cos\alpha,\quad \cos\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha$$
$$\sin\left(\frac{\pi}{2} + \alpha\right) = \cos\alpha,\quad \cos\left(\frac{\pi}{2} + \alpha\right) = -\sin\alpha$$
$$\tan\left(\frac{\pi}{2} - \alpha\right) = \cot\alpha,\quad \tan\left(\frac{\pi}{2} + \alpha\right) = -\cot\alpha$$

### $\pi \pm \alpha$

$$\sin(\pi - \alpha) = \sin\alpha,\quad \sin(\pi + \alpha) = -\sin\alpha$$
$$\cos(\pi - \alpha) = -\cos\alpha,\quad \cos(\pi + \alpha) = -\cos\alpha$$
$$\tan(\pi - \alpha) = -\tan\alpha,\quad \tan(\pi + \alpha) = \tan\alpha$$

### $\frac{3\pi}{2} \pm \alpha$

$$\sin\left(\frac{3\pi}{2} - \alpha\right) = -\cos\alpha,\quad \sin\left(\frac{3\pi}{2} + \alpha\right) = -\cos\alpha$$
$$\cos\left(\frac{3\pi}{2} - \alpha\right) = -\sin\alpha,\quad \cos\left(\frac{3\pi}{2} + \alpha\right) = \sin\alpha$$

### $2\pi \pm \alpha$（终边相同角）

$$\sin(2\pi + \alpha) = \sin\alpha,\quad \sin(2\pi - \alpha) = -\sin\alpha$$
$$\cos(2\pi + \alpha) = \cos\alpha,\quad \cos(2\pi - \alpha) = \cos\alpha$$

---

## 三、和差角公式

$$\sin(\alpha \pm \beta) = \sin\alpha \cos\beta \pm \cos\alpha \sin\beta$$

$$\cos(\alpha \pm \beta) = \cos\alpha \cos\beta \mp \sin\alpha \sin\beta$$

$$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha \tan\beta}$$

---

## 四、倍角公式

### 二倍角

$$\sin 2\alpha = 2\sin\alpha \cos\alpha$$

$$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$$

$$\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$$

### 三倍角

$$\sin 3\alpha = 3\sin\alpha - 4\sin^3\alpha$$

$$\cos 3\alpha = 4\cos^3\alpha - 3\cos\alpha$$

---

## 五、半角公式（降幂公式）

> 由 $\cos 2\alpha$ 的二倍角公式变形得到。

$$\sin^2 \frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}$$

$$\cos^2 \frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}$$

$$\tan^2 \frac{\alpha}{2} = \frac{1 - \cos\alpha}{1 + \cos\alpha}$$

$$\tan \frac{\alpha}{2} = \frac{\sin\alpha}{1 + \cos\alpha} = \frac{1 - \cos\alpha}{\sin\alpha}$$

---

## 六、和差化积公式

$$\sin\alpha + \sin\beta = 2\sin\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$

$$\sin\alpha - \sin\beta = 2\cos\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$$

$$\cos\alpha + \cos\beta = 2\cos\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$

$$\cos\alpha - \cos\beta = -2\sin\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$$

---

## 七、积化和差公式

$$\sin\alpha \cos\beta = \frac{1}{2}\bigl[\sin(\alpha+\beta) + \sin(\alpha-\beta)\bigr]$$

$$\cos\alpha \sin\beta = \frac{1}{2}\bigl[\sin(\alpha+\beta) - \sin(\alpha-\beta)\bigr]$$

$$\cos\alpha \cos\beta = \frac{1}{2}\bigl[\cos(\alpha+\beta) + \cos(\alpha-\beta)\bigr]$$

$$\sin\alpha \sin\beta = -\frac{1}{2}\bigl[\cos(\alpha+\beta) - \cos(\alpha-\beta)\bigr]$$

---

## 八、万能公式（用 $\tan\frac{\alpha}{2}$ 表示）

设 $t = \tan\frac{\alpha}{2}$，则：

$$\sin\alpha = \frac{2t}{1 + t^2}$$

$$\cos\alpha = \frac{1 - t^2}{1 + t^2}$$

$$\tan\alpha = \frac{2t}{1 - t^2}$$

---

## 九、辅助角公式

$$a\sin x + b\cos x = \sqrt{a^2 + b^2}\ \sin(x + \varphi)$$

其中 $\tan\varphi = \frac{b}{a}$，$\varphi$ 的象限由 $a, b$ 的符号确定。

---

## 十、常见特殊角三角函数值

> 记忆技巧：$\sin$ 值依次为 $\frac{\sqrt{0}}{2},\ \frac{\sqrt{1}}{2},\ \frac{\sqrt{2}}{2},\ \frac{\sqrt{3}}{2},\ \frac{\sqrt{4}}{2}$；$\cos$ 反之。

- $0\ (0^\circ)$：$\sin 0 = 0$，$\cos 0 = 1$，$\tan 0 = 0$
- $\dfrac{\pi}{6}\ (30^\circ)$：$\sin = \dfrac{1}{2}$，$\cos = \dfrac{\sqrt{3}}{2}$，$\tan = \dfrac{\sqrt{3}}{3}$
- $\dfrac{\pi}{4}\ (45^\circ)$：$\sin = \dfrac{\sqrt{2}}{2}$，$\cos = \dfrac{\sqrt{2}}{2}$，$\tan = 1$
- $\dfrac{\pi}{3}\ (60^\circ)$：$\sin = \dfrac{\sqrt{3}}{2}$，$\cos = \dfrac{1}{2}$，$\tan = \sqrt{3}$
- $\dfrac{\pi}{2}\ (90^\circ)$：$\sin = 1$，$\cos = 0$，$\tan$ 不存在
- $\pi\ (180^\circ)$：$\sin = 0$，$\cos = -1$，$\tan = 0$
- $\dfrac{3\pi}{2}\ (270^\circ)$：$\sin = -1$，$\cos = 0$，$\tan$ 不存在
- $2\pi\ (360^\circ)$：$\sin = 0$，$\cos = 1$，$\tan = 0$
